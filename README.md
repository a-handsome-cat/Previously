# Previously

**Previously** is gonna be a PastVu client with a camera, so that user will be able to make side-by-side comparison of their surroundings with old photos from said site.

**Status:** Well, I've started!

### Planned tech stack
* **Swift 5**, UIKit (Programmatic UI)
* **MapKit & CoreLocation**
* **AVFoundation** (Custom Camera)
* **MVVM** Architecture

---

**Previously** будет клиентом для PastVu с функцией камеры, чтобы пользователь мог создавать коллажи из окружающих мест и старых фотографий с сайта.

**Текущий статус:** Поехали!
